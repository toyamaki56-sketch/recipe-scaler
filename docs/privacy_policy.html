プライバシーポリシー

サービス名: Recipe Scaler  |  最終更新日: 2026-03-11

Recipe Scaler（以下「本アプリ」）をご利用いただきありがとうございます。 本ポリシーは、本アプリが取り扱う情報について説明するものです。

1. 収集する情報

本アプリは、ユーザーの個人情報・位置情報・連絡先・識別子その他いかなる個人データも収集しません。 外部サーバーへの情報送信も一切行いません。

2. 端末内に保存するデータ

本アプリは以下のデータをお使いの端末内にのみ保存します。クラウドや外部サーバーへの送信は行いません。

  • ユーザーが登録したレシピURL
  • レシピごとの人数設定
  • カートに追加した食材データ（買い物リスト）

これらのデータはアプリをアンインストールすることで完全に削除されます。

3. 第三者への情報提供

本アプリはいかなる第三者にも情報を提供・販売・共有しません。 広告ネットワーク・分析サービス・SNKとの連携もありません。

4. アプリ内 WebView でのウェブサイト閲覧

本アプリはアプリ内 WebView を使って外部のレシピサイト（クラシル・クックパッド・デリッシュキッチン等）を表示します。 ユーザーの閲覧履歴・検索履歴・クリック情報は本アプリにより収集・記録・送信されることはありません。

なお、表示先の各レシピサイトはそれぞれ独自のプライバシーポリシーに基づいて運営されています。 各サイトのデータ取り扱いについては、該当サイトのプライバシーポリシーをご確認ください。

5. 子どものプライバシー

本アプリは13歳未満の子どもを対象としておらず、当該ユーザーから意図的に情報を収集することはありません。

6. セキュリティ

本アプリはすべてのデータをユーザーの端末内にのみ保存するため、外部への漏洩リスクは最小限に抑えられています。 端末自体のセキュリティ（画面ロック等）を適切に設定されることをお勧めします。

7. ポリシーの改定

本ポリシーは必要に応じて改定されることがあります。 重要な変更が生じた場合は、アプリのアップデートを通じてお知らせします。 定期的に本ページをご確認いただくことをお勧めします。

8. お問い合わせ

本ポリシーに関するご質問・ご意見は、以下の GitHub Issues よりお寄せください。
